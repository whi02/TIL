## [오늘은 Nat Gateway에 대해 알아보았다]
Nat의 개념을 알고있다.

1. Nat를 이용하는 이유는 **사설 네트워크**에 속한 **여러개의 호스트**가 하나의 공인 IP주소를 사용하여 **internet에 접속**하기 위해서이다.
2. Internet 접속이 가능한 **Public Subnet에 NAT Gateway를 생성**해두고, **Private Subnet이 외부 인터넷으로 나아갈 경우에만 사용**하도록 라우팅을 추가하면 된다.
   (public 서브넷과 private 서브넷은 같은 VPC안에 있으면 서로 통신할수있다는 점을 이용했다)
4. NAT Gateway는 **내부에서 외부로의 접속만 가능**하고 외부에서 NAT Gateway를 이용하여 접속하는 것은 불가능하다
